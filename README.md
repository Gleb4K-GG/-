Мой проект будет вмещать в себя таблицы для заполнения необходимой информацией. Это будет служить для составления например: графика задач на день, неделю, месяц либо год. Также в программе будет встроенный калькулятор. 
- Битрикс24 (от 1С-Битрикс): Позволяет создавать задачи, вести учет времени и рассчитывать 
эффективность сотрудников.
 - Saby HRM (от Тензор): Обладает мощным функционалом для планирования графиков, 
учета рабочего времени и автоматического расчета зарплат на основе выполненных работ.
 - 1С:Зарплата и управление персоналом: Отраслевой стандарт в РФ для детального 
расчета зарплат, премий и оформления увольнений.
 - Gusto / Rippling (зарубежные): Популярные системы «все-в-одном» для малого бизнеса, 
Объединяющие кадровый учет и выплаты.
ЛидерТаск: Российский планировщик с календарем, который позволяет отслеживать 
Продуктивность и выполнение задач на день, неделю или месяц.
Shtab: Сервис для управления проектами, в котором есть встроенный трекер времени и 
возможность настройки ставок для расчета оплаты труда.
Моя программа будет в бесплатном пробном периоде до 30 дней но в ней 
будут ограничения. Пробная клиентская лицензия на 10 дней стоит 10 рублей 
также с ограничениями но с большим доступом чем в бесплатной. 
Клиентская лицензия на одно рабочее место стоит: 200 рублей.
